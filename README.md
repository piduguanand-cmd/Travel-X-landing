# TravelX Landing Page

## Project Overview

TravelX Landing Page is a modern and responsive website developed using HTML, CSS, and JavaScript. The project is designed to showcase a travel company's services and destinations through an attractive user interface.

The website includes a navigation bar, hero section, feature cards, about section, contact form, and footer. It provides a clean layout and responsive design for different screen sizes.

## Features

- Responsive Navigation Bar
- Attractive Hero Section
- Feature Cards with Images
- About Us Section
- Contact Form
- Interactive Buttons
- Form Validation Using JavaScript
- Mobile-Friendly Design
- Clean and Modern User Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Install the Live Server extension.
4. Right-click on index.html and select "Open with Live Server".
